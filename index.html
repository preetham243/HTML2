CHAPTER 1: AN INTRODUCTION

HTML, also known as the HyperText Markup language, is a DSL designed to create webpages.

In this book, we will learn how to use basic HTML.

Before we start, we have set up a place where you code.

Instructions are below:

Step 1: Download VSCODE.

Step 2: Download the extension live server.

Step 3: Create a new file and name it "index.html"(you can name it other things, but index is needed to publish a site.)

Now that we have set up VSCODE, we can start learning some HTML elements and there purpose!


<!DOCTYPE HTML> - Used at the beginning of every HTML file. It's a MUST.
<html> - The container of all HTML code. Although browsers are forgiving, not having it violates the specification.
<h1> - The largest heading. Its usually the title of the page.
</tag> - Used to close a tag. just replace "tag" with a real tag.
<head> - Used to add in the JS and CSS links, the favicon, and the title.
<body> - Used to add in the regular stuff, like <p> or <h1>!

Alright! Now that we have learned a few elements, lets learn how to use them! turn the page to find out!
